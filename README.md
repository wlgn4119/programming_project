# programming_project

## 숫자 야구 게임 (Number Baseball Game)

C언어로 구현한 콘솔 기반 숫자 야구 게임입니다.

---

# 1. Project Overview (프로젝트 개요)

* **프로젝트명:** 숫자 야구 게임
* **개발 언어:** C
* **개발 환경:** Visual Studio Code
* **프로젝트 설명:**

  * C언어 수업에서 학습한 입출력, 조건문, 반복문, 함수, 배열 등의 개념을 활용하여 제작한 콘솔 게임입니다.
  * 컴퓨터가 생성한 중복 없는 숫자를 플레이어가 추리하여 맞추는 게임입니다.

# 3. Game Rules (게임 규칙)

1. 컴퓨터가 중복되지 않는 숫자로 정답을 생성합니다.
2. 플레이어는 숫자를 입력하여 정답을 추측합니다.
3. 결과는 다음과 같이 판정됩니다.

| 판정     | 설명               |
| ------ | ---------------- |
| Strike | 숫자와 위치가 모두 일치    |
| Ball   | 숫자는 일치하지만 위치가 다름 |
| Out    | 일치하는 숫자가 없음      |

예시 (정답 : 1 3 5)

| 입력    | 결과       |
| ----- | -------- |
| 1 2 4 | 1 Strike |
| 5 1 2 | 2 Ball   |
| 2 4 6 | Out      |

# 4. Key Features (주요 기능)

### 게임 시작

* 난이도 선택 가능
* 컴퓨터가 랜덤 정답 생성

### 난이도 선택

| 난이도 | 자리 수 | 기회 |
| --- | ---- | -- |
| 쉬움  | 3자리  | 9회 |
| 보통  | 3자리  | 7회 |
| 어려움 | 4자리  | 7회 |

### 입력 검증

* 숫자가 아닌 문자 입력 방지
* 범위를 벗어난 숫자 입력 방지
* 중복 숫자 입력 방지

### 게임 기록

* 시도 기록 출력
* Strike / Ball / Out 결과 출력

### 최고 기록

* 난이도별 최고 기록 저장
* 게임 종료 후 최고 기록 확인 가능
